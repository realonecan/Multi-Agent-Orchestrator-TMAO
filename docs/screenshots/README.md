## Creating Screenshots

### Windows
```powershell
# Use Windows Terminal built-in screenshot
# Or use Snipping Tool
```

### Linux
```bash
# Use gnome-screenshot
gnome-screenshot -w

# Or use scrot
scrot -u
```

### macOS
```bash
# Use built-in screenshot
cmd+shift+4
```

## File Naming Convention

- `command-center-main.png` - Main dashboard view
- `command-center-dark-neon.png` - Dark Neon theme
- `command-center-solarized.png` - Solarized theme
- `command-center-chat.png` - Chat panel closeup
- `command-center-metrics.png` - Metrics panel closeup
- `command-center-orchestration.gif` - Full orchestration demo
- `command-center-theme-switch.gif` - Theme switching demo

## Usage in Documentation

Reference screenshots in markdown:

